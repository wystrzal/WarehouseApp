## Setup
To run API open Warehouse.API.sln with Visual Studio, change from IIS Express to Warehouse.API and press F5.

To run SPA open Warehouse folder with Visual Studio Code and 
`cd Warehouse-SPA` 
`npm install`
`ng serve`

Open http://localhost:4200
